```sh

```

Command line usage requires that you have the necessary software installed.  See
[the instructions](https://github.com/martinthomson/i-d-template/blob/main/doc/SETUP.md).

## Regenerating Examples

The draft markdown uses HTML comment markers
(`<!-- begin:... ; see README for regeneration instructions, do not edit -->` /
`<!-- end:... -->`) to delimit auto-generated sections. Content between
matching begin/end pairs is overwritten by the script and should not be edited
directly in the markdown.

The JWKs, JWE test vectors, aggregate vector file, and Test Vectors appendix
are generated by scripts in the `scripts/` folder. The algorithm suites are
defined in `scripts/algorithms.js`.

The generated outputs must be committed to the repository. After modifying the
algorithm set or vector generation logic, run:

```sh
# requires Node.js >=24.7.0
npm install
node --run update
```

To force regeneration of raw examples regardless:

```sh
node --run force-update
```
